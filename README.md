# qu1zz3r
Quizz maker application, made in python.

Features:
 - Creating quizzes.
 - Max 25 questions.
 - Every question has a max of 6 possible answers.
 - Set points for right answers.
 - Quizzes are stored as files.
 
In the future (maybe):
 - Login.
 - Ability to edit old quizzes.
 - Game system - 4 players max.
